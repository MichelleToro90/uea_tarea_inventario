# 📦 uea_tarea – Sistema de Gestión de Inventarios (GUI + SQL)

Proyecto académico desarrollado en Python que implementa un **Sistema de Gestión de Inventarios** con:

- ✅ Interfaz gráfica (Tkinter)
- ✅ Persistencia de datos en **SQLite (SQL)**
- ✅ Manejo de excepciones
- ✅ Arquitectura por módulos
- ✅ Ejecución desde PowerShell / VS Code

---

## 🎯 Objetivo del Proyecto

Desarrollar un sistema que permita:

- Agregar productos  
- Actualizar productos  
- Eliminar productos  
- Listar productos  

Guardando la información en una base de datos SQL local y mostrando una interfaz gráfica amigable para el usuario.
